This is a Chess game that uses an algorithms which is, MinMax, NegaMax, NegaMaxAlphaBeta for offline opponent that using a basic algorithm.
You may play this game with your friend, or if you want play it alone, you can play it with an algorithm

(RUN INSTRUCTION)
Open your Visual Studiio Code
Open this file
Run the program from the ChessMain.py
if you want to play single player, you should switch the (playerTwo = true) in line 37 from the ChessMain.py
and if you want to play with different types of algorithms you can select the algorithms in line 57 until 59, select one of the algorithm and delete the hashtag on one of the algorithms

(DEPENDENCIES)
The code must use one  of the algorithms if the player want to play it as singleplayer.
